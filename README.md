# カスタムグラインドページ サンプル

## 内容
- コーヒーマシンを選択すると、おすすめの挽き具合が動的に切り替わります。
- HTML / CSS / JavaScriptのみで動作します。
- 外部ライブラリは使用していません。
- GitHub Pagesでそのまま公開できます。

## ローカル確認
`index.html` をブラウザで開いてください。

## GitHub Pages公開手順
1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルをリポジトリ直下へアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Build and deployment` の Source を `Deploy from a branch` にします。
5. Branchを `main`、フォルダを `/(root)` にして保存します。
6. 数分後に公開URLが表示されます。

## 注意
- 料金はサンプル値です。
- 本番導入時はFutureshopの商品ページURLやカート導線へ差し替えてください。
- 楽天・Yahoo・Amazonへ転用する場合は、各モールのHTML・JavaScript制限に合わせた静的版が必要です。
